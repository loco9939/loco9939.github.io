Grid Layout 알아보기

2023/08/09 카테고리 Daily

푸터를 만들 때, Grid를 사용하여 설계를 해보자.

  • 전체적인 레이아웃은 grid를 사용하고 내부의 세부적인 레이아웃은 flex를 사용한다.
  • grid 내부의 크기를 grid-area로 지정할 수 있다.
  • grid-area: 열,행
  • grid는 margin-collapse 일어나지 않는다.
