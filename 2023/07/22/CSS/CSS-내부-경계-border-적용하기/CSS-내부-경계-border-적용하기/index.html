CSS 내부 경계(border) 적용하기

2023/07/22 카테고리 CSS

내부 경계(border) 효과 주기

문제

CSS로 작업을 하다가 경계가 검정색 바탕 부분과 border에 약간의 단차가 생기는 문제가 있었다.

마크업 구조는 다음과 같다.

  • h2 태그에 배경색이 검정색이다.
  • h2 태그와 형제태그인 div 태그에 보더를 주었더니 단차가 생겼다.

해결방법

  • box-shadow 효과를 inset으로 주어서 보더 처럼 보이도록 구현했다.
