<오늘부터 CSS 마스터 11일차> combinators(셀렉터 조합)

2023/02/27 카테고리 CSS

combinators 셀렉터 조합

4가지의 셀렉터 조합이 있다.

  • 자손(descendant) 셀렉터 (space)
  • 자식(child) 셀렉터
  • 인접한 형제 셀렉터 (+)
  • 두루두루의 형제 셀렉터 (~)

descendant 셀렉터
