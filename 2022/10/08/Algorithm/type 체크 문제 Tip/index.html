2022/10/08 카테고리 Algorithm

문제

입력값에 {}, [], null이 들어가면 출력값으로 type을 반환하도록 하는 함수를 만들어라

내 코드

  • 입력값에 object 타입만 받을 수 있도록 에러처리를 해주었다.
  • if, else if 문으로 조건에 따라 반환값을 반환해주었다.

새로운 코드

  • call 메서드를 사용하여 data 값을 바인딩 toString() 메서드에 바인딩 해주었다.
